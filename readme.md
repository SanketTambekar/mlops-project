# 💼 Salary Prediction ML Project

## 📌 Objective
Predict salary category (>50K or ≤50K) using ML.

## 📊 Dataset
Small synthetic dataset with:
- Age
- Experience
- Education

## ⚙️ Tech Stack
- Python
- Pandas
- Scikit-learn
